About me

Hello! I am hypews and I am a Java developer where I specialize in the Spigot API. I have made basic webpages, to full fledged applications using various APIs - my main one being the Lyfe Discord Bot which is now abandoned. I've been writing in Java for around a year now and I am very familiar with the SpigotAPI. I have made a few Minecraft plugins which are public on SpigotMC
